## Usage :smile:

1. Just follow [official instruction](https://developer.chrome.com/docs/extensions/) and install this extension
1. Just select text and trigger shortcuts

## Shortcuts :handshake:

| shortcut          | action                         |
|-------------------|--------------------------------|
| Ctrl+Shift+E (macOS: Command+E)     | Emotionalize text and Copy     |
